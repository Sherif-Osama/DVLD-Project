# DVLD Project

Driving & Vehicle License Department System

## 📌 Description
A Windows Forms application built using C# that manages:
- People
- Drivers
- Licenses
- Applications

## 🧱 Architecture
- Business Layer
- Data Access Layer
- DTO Layer
- UI (DVLD)

## 🛠 Technologies
- C#
- .NET Framework
- Windows Forms
- SQL Server

## 🚀 How to Run
1. Open DVLD.sln in Visual Studio
2. Build Solution
3. Run (F5)
