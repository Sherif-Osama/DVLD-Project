# DVLD Project

Driving & Vehicle License Department System

## 📌 Description

A desktop application built using C# and Windows Forms that simulates a real-world Driving & Vehicle License Department system.
It allows managing people, drivers, licenses, applications, and scheduling driving test appointments, all within a structured multi-layer architecture (Business, Data Access, DTO, and UI).

## 🧱 Architecture

* Business Layer
* Data Access Layer
* DTO Layer
* UI (DVLD)

## 🛠 Technologies

* C#
* .NET Framework
* Windows Forms
* SQL Server

## 🚀 Features

* Manage people and drivers
* Issue and renew licenses
* Manage applications and driving tests
* Schedule and manage test appointments
* Advanced search and filtering

## ▶️ How to Run

1. Open DVLD.sln in Visual Studio
2. Configure the database connection (SQL Server)
3. Build Solution
4. Run (F5)
